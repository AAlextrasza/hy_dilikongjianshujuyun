ArcGIS 10.7正式版已于2019年3下旬重磅发布。新版本平台体系新增众多产品及业务应用， 助力科学分析与智能决策；平台三维、大数据、人工智能等核心能力大大增强，为用户打造了 更智能、更强大的地理空间云平台。同时ArcGIS 10.7提供了基于用户类型的授权机制，极其 灵活的方式使得地理空间云更好的服务组织机构的每一类业务人群。

一、新增产品及应用

1. Notebook Server，全新数据科学服务器产品

ArcGIS Enterprise 10.7中新增全新服务器角色—ArcGIS Notebook Server。它是面向数据科学家、数据分析师等角色所打造的在线交互式python开发环境，可以在线创建、共享、运行python脚本，基于空间数据进行数据建模、数据挖掘与分析、数据管理、ArcGIS平台运维等工作。

其核心是ArcGIS API for Python和ArcPy，此外还集成了丰富的Python开源分析、统计以及机器学习库，如TensorFlow、 scikit-learn和 fast.ai等，提供集成化的环境，结合ArcGIS平台海量的地理数据资源，让用户更关注于解决数据分析问题本身。

2. 全新地理赋能的业务型应用，覆盖各类业务流

ArcGIS平台包含几十种即拿即用的通用型Apps，如Map Viewer、ArcGIS Maps for Office、配置型Apps、地图故事等，使用户可以方便的通过Apps访问平台资源和能力。ArcGIS 10.7又新增众多面向专业业务流的Apps，为组织机构各类业务人员提供地理赋能的专业方案。

3. ArcGIS Excalibur，即拿即用的在线影像工作流

ArcGIS Excalibur是全新的Web应用程序，能够帮助用户快速发现、分析、报告及分享影像中包含的信息与价值，为组织机构的影像管理人员、影像分析师、数据分析师提供快速需求响应。

>ArcGIS Excalibur与平台无缝集成，可以依靠Image Server强大的栅格函数功能对影像数据进行实时处理和动态增强，提供工程管理能力，可方便组织影像工作流所需的所有资源，同时还允许编制、发布和共享影像工作报告及静态、动态信息产品，分发给最终用户。

4. ArcGIS Indoors，完整的室内外定位导航方

ArcGIS Indoors是一套完整的室内外导航解决方案, 它集室内地图制作、室内定位、室内外路径导航、协作分享于一体，可广泛应用于零售和商业场所，机场，医院，校园等场景。

ArcGIS Indoors包括两部分：桌面端的内容制作工具、Web端和移动端导航应用。 其中桌面端工具可将室内平面图，如CAD数据，批量自动化转换为ArcGIS室内信息模型，并依据算法科学生成室内通行路网，与传统手工绘制室内地图、手工绘制路网的方式相比，更加科学、高效，节约人力成本；室内数据制作完成后，便捷分享到云端，用户随时随地开启导航应用，让室内外一体化导航触手可及。

5. ArcGIS Urban，城市规划解决方案

ArcGIS Urban是一款全新的Web端三维应用，提供完整的城市规划解决方案，用于协调城市开发和规划工作流程。基于Urban，城市管理者、开发商、市民可在三维视图中了解城市规划和建设项目情况；辅助设计人员进行建设项目选址分析、规划方案调整及指标分析，同时可通过标准化流程评估、审核每个开发项目；市民可以提供建设性意见。

>ArcGIS Urban将规划部门、开发商、市民紧密联系在一起，共同建设健康、可持续发展的活力城市。

6. Tracker for ArcGIS，开启外业人员位置追踪能力

Tracker for ArcGIS是全新推出的具有位置追踪能力的应用，它包含三个关键组件：位置追踪服务、Web端追踪视图和移动端追踪应用。管理者可以基于Web端追踪视图查看和管理用户的实时位置，对历史轨迹进行分析，管理外业人员工作；外业人员可通过移动端App进行轨迹的实时记录和上传，支持离线使用。

Tracker for ArcGIS非常适用于公共安全、应急、外业作业、VIP客户安保等领域，如在体育比赛、音乐会等重大事件上用它来追踪经理人员的实时分布以确定哪些区域存在警力不足的隐患；在应急救灾和恢复阶段，及时了解救援人员分布；外业人员是否有按照规定的区域和路线保质保量的完成巡检任务等。

二、核心能力提升

1.三维与BIM，亿级数据高效浏览

ArcGIS在BIM和倾斜摄影等数据支持、三维分析、大数据量展示性能和效率等方面都有突破性进展。ArcGIS 10.7进一步增强和完善。

BIM支持：作为全球唯一一款原生支持Revit的产品，10.7对BIM的支持从底层到应用全面优化。OGC I3S数据标准专门针对BIM数据增加全新的Building类型规范，Web端新增Building Explorer微件，可方便地分专业、分类别查看BIM数据。

2.大数据，丰富的大数据分析能力

ArcGIS 10.7空间大数据在产品功能、扩展性、性能、运维监控等方面进行了整体革新，新增7种矢量大数据分析工具，包括裁剪数据、融合数据、合并数据、描述数据集、构建多值段网络、随机森林分类、广义线性回归等。一方面高效支撑传统业务需求的分析和运算，如国土行业中的土地利用现状分析、叠加分析、冲突检测、变化监测等计算；另一方面，集成了机器学习能力，能够实现历史数据洞察挖掘和分析预测，如房价预测、餐馆收入预测等内容。

ArcGIS GeoAnalytics Server 10.7，功能增强、性能提升、安全稳定、全面革新，具备海量空间数据的高效运算和洞察挖掘能力，是一个全面、完整的空间大数据分析产品。

3.影像，五大关键能力搬到云端

ArcGIS 10.7影像能力全面向着云端发展，在新版本中“云”的理念已经无处不在，为用户带来全新的体验模式。

云存储支持：ArcGIS Pro、Image Server均支持主流的面向对象存储和分布式存储，包括AWS S3、Azure Blob、阿里云OSS（Alibaba）、华为云OBS（Huawei）以及Web HDFS。影像数据可以直接云端存储，在桌面及服务器产品中读写、浏览与分析。

云端影像管理与发布：ArcGIS 10.7支持在Portal门户中在线创建影像图层，数据可以来自云端存储或共享文件系统。

云端地图生产：OrthoMaker产品可在线生产正射影像、DSM、DTM等数据产品。ArcGIS 10.7进一步基于控制点提升数据产品精度。

ArcGIS 10.云端影像分析工作流：ArcGIS 10.7在Portal门户中新增支持影像分析工作流自定义编辑器，在线提供近150种栅格处理与分析函数，用户可基于可视化窗口自定义分析流程。

ArcGIS 10.span style="font-weight:bold;">云端发现与探索：用户可利用Portal门户的智能检索能力与可配置应用快速发现、探索影像数据。ArcGIS 10.7新增ArcGIS Excalibur产品，为用户提供了即拿即用的完整在线影像工作流

总结

在数字化转型的今天，数据驱动、智能决策是业务成功的关键，ArcGIS 10.7无论是新增的数据科学计算服务器、各种覆盖业务流的专业型应用，还是三维、大数据、人工智能、影像等核心能力的提升，都使整个平台智能化分析、辅助决策的能力更加强大，必定能在业务增长的道路上祝您一臂之力！
